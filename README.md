# Краткое описание функционала

Реализованная нами программа позволяет вести учет расходов по дням и категориям.
Основные функции (с использованием пройденных тем):
Добавлять расходы (день, сумма, категория);
Считать сумму за любой период (префиксные суммы);
Показывать день с самой большой тратой (линейный поиск);
Показывать топ категорий по сумме трат (сортировка вставками);
Отменять последнее добавление (стек);
Показывать дерево трат (месяц - дни - категории).

Для запуска: main.py
